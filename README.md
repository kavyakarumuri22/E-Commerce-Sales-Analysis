# E-Commerce Sales Analysis

## Pluto Academy Data Analytics Internship – Project 1

### Project Overview
This project analyzes the Olist Brazilian E-Commerce dataset to identify sales trends, top-performing product categories, regional performance, customer review patterns, and business insights.

### Objectives
- Perform data cleaning and preprocessing
- Analyze monthly sales trends
- Identify top revenue-generating categories
- Compare regional performance
- Analyze customer review scores
- Build a management dashboard
- Provide business recommendations

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

### Project Files
- E-Commerce_Sales_Analysis.ipynb
- dashboard.png

### Key Insights
- Identified the highest revenue-generating product categories.
- Analyzed monthly sales trends.
- Compared regional performance.
- Evaluated customer review score distribution.
- Suggested data-driven business recommendations.

### Dataset
Olist Brazilian E-Commerce Dataset (Kaggle)
